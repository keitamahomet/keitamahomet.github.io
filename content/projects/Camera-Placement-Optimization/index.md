This project focuses on solving a combinatorial optimization problem related to the optimal placement of surveillance cameras in a building. The objective is to determine the minimum number of cameras required to monitor all corridors while ensuring full coverage of the infrastructure.

The problem is modeled using mathematical optimization techniques commonly used in operations research. Each corridor must be covered by at least one camera, and the optimization model determines the most efficient configuration that satisfies this constraint while minimizing the total number of cameras installed.

The model is implemented in the R programming language using optimization techniques designed to handle combinatorial decision problems. The mathematical formulation translates the infrastructure coverage problem into a structured optimization model where decision variables represent potential camera placements and constraints ensure complete corridor surveillance.

By solving the model, the optimal camera configuration is identified, guaranteeing that all corridors are monitored with the smallest possible number of cameras. This approach demonstrates how prescriptive analytics can be used to solve real-world infrastructure and security planning problems.

This type of optimization framework is widely applied in areas such as facility security planning, network monitoring, infrastructure coverage, and resource allocation. The project highlights how optimization models can support efficient decision-making when dealing with complex spatial coverage problems.
