This project focuses on developing a mathematical optimization model to determine the optimal allocation of investments across multiple financial opportunities while respecting a set of predefined constraints. Using techniques from prescriptive analytics and operations research, the model aims to maximize expected returns while considering practical limitations such as budget constraints, allocation limits, and investment diversification requirements. The problem is formulated as a mathematical programming model and implemented using analytical tools in R. The approach demonstrates how optimization techniques can support structured decision-making by identifying the most efficient distribution of available resources. Through this project, the model evaluates different allocation scenarios and identifies the investment strategy that yields the best financial outcome given the constraints. This type of optimization framework is commonly used in financial planning, portfolio management, and strategic resource allocation. Overall, the project illustrates how prescriptive analytics can transform complex decision problems into solvable mathematical models, providing actionable insights for decision-makers and improving strategic investment planning.
