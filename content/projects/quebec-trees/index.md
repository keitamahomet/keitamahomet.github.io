This project focuses on the exploratory analysis of a dataset describing tree populations across different regions of Quebec.

Using descriptive statistics and visualization techniques, the analysis aims to identify patterns in tree species distribution, environmental characteristics, and regional variations. The project demonstrates how exploratory data analysis can be used to better understand ecological datasets and uncover meaningful insights.

The analysis was conducted using the R programming language, leveraging statistical summaries and graphical visualizations to explore the structure and characteristics of the dataset.
